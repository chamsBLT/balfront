About our project

Let's talk cloud



Consomi Tounsi is a cloud based product management platform dedicated for selling 100% Tunisian products.

The first goal of the project was to provide virtualized computing resources on demand. These resources are servers, storage spaces or containers.

Also, the project proposes a solution to minimize provisioning time, eliminate repetitive tasks and reduce mistakes & errors risk.

Finally, the project provides the deployment environment required to run the web application with centralized management and orchestration for all the deployment environments.
